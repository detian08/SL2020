All In One Cancel Feature in Odoo

This app helps user to cancel all order in odoo like sale, purchase, invoice, picking, inventory adjustment and landed cost. In short for this module will allow user to undo records in some cases in Odoo ERP

Cancel Sales Order

User can cancel validated sales order, delivery order, customer invoice. And Journal entry and stock journal entry will automatically deleted.

Cancel Purchase Order

It is possible to cancel confirmed purchase order, incoming shipment, vendor bill, journal entry, and stock journal entry will automatically deleted.

Cancel Stock Picking

User can cancel validates stock picking, product quantity will reset on canceling and stock journal entry will deleted, user can set to draft order.

Cancel Landed Cost

Cancel validates landed cost and journal entry and account moves related to landed cost automatically deleted.

Cancel Inventory Adjustment

User can cancel validated inventory adjustment, product on hand quantity effected and journal entry of inventory adjustment will deleted.

Cancel Related Operations

On canceling order related operations like picking/delivery, invoice/bill will cancelled. Stock move, journal entry, and stock journal entry will deleted, and product quantity will updated.

Description

Cancel Sale Order

User can cancel sale order even if in "Sale Order" state.



On canceling sales order related invoice and delivery order will automatically cancelled. Journal entry, and stock journal entry will deleted, after canceling sales order user can move the order to "Set to Quotation".


Cancel Purchase Order

User can cancel purchase order even if in "Purchase Order" state.



On canceling purchase order related shipments, and vendor bill will automatically cancelled. Journal entry, and stock journal entry will deleted, after canceling purchase order user can move the order to "Set to Draft"


Cancel Stock Picking

User can cancel stock picking even if in "Done" state.



When user cancel picking order than product quantity will updated and stock journal entry will deleted, user can "Set to Draft" the picking.


Cancel Inventory Adjustment

User can cancel inventory adjustment even if "Validated".



On canceling inventory adjustment on hand quantity on product and lot/serial number will effected and related stock journal entry will deleted, user can "Set to Draft" inventory adjustment after cancel.


Cancel Landed Cost

User can cancel landed cost even if in "Posted" state.



On canceling landed cost stock journal entry and account moves related to landed cost automatically deleted, user can also "Set to Draft" landed cost after cancel.
